# Homebrew Tap

Homebrew formulae for [schmitthub](https://github.com/schmitthub) projects.

## Install

```sh
brew install schmitthub/tap/clawker
```

Or tap first, then install:

```sh
brew tap schmitthub/tap
brew install clawker
```

## Available packages

| Formula | Description |
|---------|-------------|
| [clawker](https://github.com/schmitthub/clawker) | Development containers for AI coding agents |

## How it works

Formulae are automatically updated by [GoReleaser](https://goreleaser.com) on each release. Do not edit formula files directly.
